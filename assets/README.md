# Assets 目录

此目录用于存放应用的静态资源文件。

## 需要的文件

### icon.png
应用图标文件
- 推荐尺寸：1024x1024 像素
- 格式：PNG

### splash.png
启动屏图片
- 推荐尺寸：1242x2436 像素
- 格式：PNG

### adaptive-icon.png
Android 自适应图标
- 推荐尺寸：1024x1024 像素
- 格式：PNG

## 临时方案

在添加正式资源文件之前，你可以：
1. 使用 Expo 默认的资源文件
2. 从 https://iconscout.com/ 或其他免费资源网站下载图标
3. 使用在线工具生成图标
